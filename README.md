# GMC-300 Log Viewer

Online log viewer for GQ GMC-300 Geiger Muller Counter. Upload CSV data logs to visualize radiation measurements (CPM, µSv/h, mR/h) with interactive charts.

**Live demo:** https://andkom.github.io/gmc-viewer/

## Features

- Interactive time-series chart powered by Apache ECharts
- Three measurement units: CPM, µSv/h, mR/h
- Location flags support
- Zoom, pan, and save-as-image
- Predefined example dataset (Chernobyl Exclusion Zone, Oct 2012)

## Usage

Upload a CSV data log exported from your GMC-300 device, or try the built-in example via the [Chernobyl dataset](https://andkom.github.io/gmc-viewer/#chernobyl).
